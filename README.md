# ZeroG5.tech
This is my personal site where you can find some info about me like my skills, interests or ways to contact me. It is solely made for entertainment, but in the past it had a bit different function as it was a part of a bigger project, where I was testing cross-site scripting. 
# Components
- background with particles
- terminal
# The terminal
The main part of the site, which is solely dependent on javascript. It is a linux-style terminal with all the basic commands and my custom ASCII art. There are, of course, some specific commands for certain outcomes. It is programmed to be interactive, but only with limited responses that have been pre-written for each command. It looks real, but it is entirely sandboxed and runs only on the client side.

# Structure
My goal was to create a strucuture similar to linux filesystem, that said, here is the whole structure of files, including the hidden ones:
```
-/home
 |- /zerog
  |- about.txt
  |- skills.txt
  |- projects.txt
  |- contacts.txt
  |- locked.sh
  |- -
  |- .hidden.txt
-/var
 |- /log
  |- session.log
  |- syslog
-/dev
 |- null
 |- random
 |- zero
-/bin
 |- cat
 |- bash
 |- ls
 |- grep
 |- chmod
-/etc
 |- motd
 |- passwd
 |- hostname
 |- os
-/root
 |- config.txt
 |- .bashrc
 |- .vimrc
 |- .bash_history
```
Feel free to use this site as you want.
